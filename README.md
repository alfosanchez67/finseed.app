# 🌱 FinSeed — Educación Financiera Familiar

> Planta tu semilla financiera y observa cómo crece de manera individual o junto a tu familia.

FinSeed es una plataforma web de educación financiera gamificada para familias chilenas. Los padres invierten para sus hijos y estos aprenden finanzas mediante juegos interactivos, misiones diarias y un sistema de progresión motivador.

---

## 🎯 El Problema

El 78% de los jóvenes chilenos no recibe educación financiera formal. Las familias no tienen herramientas para enseñar finanzas de forma práctica y entretenida. Los productos financieros existentes están diseñados para adultos, dejando fuera a la próxima generación de inversores.

---

## 💡 La Solución

FinSeed conecta a padres e hijos en una experiencia financiera compartida:

- **Padres**: Gestionan portafolios, crean metas familiares y monitorean el progreso educativo de sus hijos.
- **Hijos**: Aprenden finanzas jugando con XP, rachas, vidas, misiones diarias y un árbol de conocimiento progresivo.

## 🏗️ Arquitectura Técnica

- **Frontend:** HTML5 + CSS3 + JavaScript Vanilla — Single Page Application
- **Backend:** Supabase (PostgreSQL + Auth + API REST)
- **Hosting:** Vercel con auto-deploy desde GitHub
- **Precios mercado:** Twelve Data API
- **PWA:** Manifest + instalable en iOS y Android

## 🎮 Sistema de Juegos

**Introducción obligatoria** → desbloquea 7 caminos temáticos en paralelo:

- 🐷 Ahorro y Presupuesto
- 📈 Inversión y Bolsa
- 🪙 Criptomonedas
- 🏠 Bienes Raíces
- 🚀 Emprendimiento
- 🌍 Economía y Macro
- 📑 Impuestos y Estado

**Formatos de juego:**
- Opción múltiple con explicación objetiva
- Simulaciones con cálculos reales
- 🃏 Swipe decisions
- 🎯 Simulador de presupuesto con sliders
- 🕵️ Detective de Estafas

## 👨‍👩‍👧 Flujo de Usuario

**Padre:**
1. Registro → elige rol Padre
2. Agrega hijos (nombre, edad)
3. Distribuye fondos a hijos
4. Monitorea progreso educativo y logros
5. Compra/vende acciones en portafolio familiar

**Hijo:**
1. Registro → elige rol Hijo → vinculación automática con padre
2. Explora árbol de juegos → desbloquea caminos
3. Completa misiones diarias → gana XP y vidas
4. Ve su portafolio y solicita retiros
5. Chatea con la familia
